## 问题描述说明

::: info
Mybatis-Plus中UpdateWrapper在set设置Oracle数据库值为null报错，
原因是JdbcType为Other，不被支持。而在Mysql数据库中此操作正常。
解决方案包括修改全局配置为field-strategy:IGNORE或针对特定字段设置jdbcType。
:::

## 解决方案

::: tip
更改配置中的jdbc-type-for-null 为 'null'
:::

```yaml
mybatis-plus:
  configuration:
    jdbc-type-for-null: 'null'
```
